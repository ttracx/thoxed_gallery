# Security Policy

## Reporting

Report suspected vulnerabilities through the owner-approved private THOX.ai LLC security channel or GitHub private vulnerability reporting when enabled for this repository.

Do not disclose suspected vulnerabilities publicly until THOX.ai LLC has triaged and remediated them.

## Scope

This policy covers THOX-specific code, configuration, documentation, scripts, product definitions, and release artifacts in this repository. Third-party and upstream components remain subject to their original maintainers' vulnerability handling processes.
